## Features

| Feature | Description |
| --- | --- |
| **Clothing System** | 10 wearable slots with gender-specific drawables applied in real time |
| **Rarity System** | 5 configurable tiers (Common → Legendary) with custom colours, gradients and animations |
| **Utility Slots** | 8 dedicated quick-access slots with SVG icons (armor, backpack, phone, and more) |
| **Armor System** | Two armor types with job restrictions, plate tracking, damage rates and repair items |
| **Backpack System** | Three backpack sizes, each opening as a separate inventory with slot and weight limits |
| **Crafting System** | Placeable workbenches, XP progression, blueprint-locked recipes, two bench tiers |
| **Storage System** | Placeable crates and safes with job-group permissions |
| **Shop System** | Multi-location shops with target zones, blips and category filtering |
| **Robbing System** | Full robbery UI — steal from and place items into another player's clothing and utility slots |
| **Blueprint System** | Admin-granted blueprint items that unlock advanced crafting recipes |
| **UI Themes** | 30 built-in colour themes, switchable with a single config line |
| **Throw / Place / Pickup** | Throw items as world props, place storage and benches, pick them up |
